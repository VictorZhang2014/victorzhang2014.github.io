Python的科学计算包 - NumPy

  2017-12-24     AI     Python, Numpy

NumPy Documentation Officially是一个第三方的Python包，用于科学计算。这个库的前身是1995年就开始开发的一个用于数组运算的库。经过了长时间的发展，基本上成了绝大部分Python科学计算的基础包，当然也包括所有提供Python接口的深度学习框架。)

NumPy是用Python编写的，用于科学计算的基础包，包含了以下四种特性：

  • 对于处理N维数组对象非常强大
  • 复杂（广播）函数 sophisticated (broadcasting) functions
  • 便于集成C/C++/Fortran代码
  • 支持线性代数，博里叶变换，随机数
